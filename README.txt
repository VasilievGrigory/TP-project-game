На данный момент сделана часть игры по созданию персонажей: вы можете выбрать себе 3 героев, которыми в дальнейшим будете играть.
Проект может будет запущен по средствам make:
1) скачать файлы с репозитория
2) скачать библиотеку <ncurses.h> (sudo apt-get install livbncursesw5-dev)
3) make game
4) ./game
